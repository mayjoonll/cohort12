# cohort-12

코호트 기반 실습 학습 저장소입니다.  
매일 하나의 테스크를 수행하고 결과를 남깁니다.

---

## 참여 방법

1. 이 저장소를 **fork**
2. 각 테스크 폴더에서  
   `submissions/본인이름.py` 파일 생성
   (예: `day1-localserver/submissions/kms.py`)
3. 예제 코드 복사 후 일부 수정
4. 커밋 & 푸시
5. **PR 1개 생성** (템플릿 사용)

- 제출: **Pull Request**
- 질문: **Issue**
- README 수정: **금지**
